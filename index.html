Custom webfont icons

A quick demo demonstrating how to create a custom icon font with SVGs, for use with SCSS.

The icon font is named based on the project name specified in package.json

SCSS way:

                    
                        .my-element:before { @include custom-webfont-icons(facebook) }

HTML class way:

                    
                        <div class="custom-webfont-icons-icon-twitter">Twitter</div>

Result:

  • Twitter
